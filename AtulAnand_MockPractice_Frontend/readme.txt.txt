Project Name: Event Management Website - Mock Test 1 (Fullstack MERN)

Developed By: Atul Anand

Description:
This is a responsive event management website showcasing upcoming events, event highlights, and a registration form. 
It uses HTML5, CSS3, and Bootstrap for layout and styling, and JavaScript to dynamically load event data from a JSON file.

Folder Structure:
- UserStory1/
    - index.html        → Homepage with event highlights and registration form
    - style.css         → Custom styling
- UserStory2/
    - dashboard.html    → Dashboard displaying all events
    - script.js         → Fetches and displays events from events.json
    - events.json       → Contains event details in JSON format
- Screenshots/         → Contains screenshots for report submission

Setup Instructions:
1. Download or extract the project folder.
2. Open index.html (inside UserStory1) in your web browser to view the homepage.
3. Click “Register” to navigate to the dashboard page (UserStory2/dashboard.html).
4. Make sure events.json is in the same folder as script.js and dashboard.html.

Dependencies:
- Bootstrap 5 (via CDN)
- Font Awesome (optional)
- No external dependencies or installation required.
